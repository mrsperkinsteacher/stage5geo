LESSON 19

Grid and Area Referencing

Learning intention: we are learning to give four-figure area references and six-figure grid references.

Lesson outline

  1. Teach (10 min): eastings before northings (“along the corridor, up the stairs”); four-figure AR for a square, six-figure GR for a point.
  2. Independent work (40 min): textbook/Atomi exercises; final interviews.
  3. Check (5 min): quick-fire class quiz — teacher calls a feature, students give the GR.

Main task instructions

  1. Give four-figure area references for five features and six-figure grid references for five points on the supplied map extract.

LESSON 20

Interpreting Graphs

Learning intention: we are learning to read, describe and interpret line, column, sector and compound graphs.

Lesson outline

  1. Teach (10 min): the describe–explain routine: identify max/min, overall trend, anomalies; use numbers in every sentence.
  2. Independent work (40 min): mixed graph interpretation set (population and migration data — recycles unit content as skills practice).
  3. Check (5 min): mark from key.

Main task instructions

  1. Complete the mixed graph set: one line graph (population growth), one column graph (interstate migration), one sector graph (migration intake), one compound graph.

External links

  • Atomi — interpreting graphs module

LESSON 21

Skills Consolidation

Learning intention: we are consolidating all six skills before returning to unit content.

Lesson outline

  1. Circuit (45 min): six-station skills circuit (climate graphs, scale, gradient, contours, grid references, graphs) — 7 minutes per station, answers checked at each rotation.
  2. Wrap (10 min): students self-assess against the six skills (traffic light) and nominate one skill for revision before the yearly exam.

Main task instructions

  1. Complete the six-station circuit sheet; traffic-light self-assessment; catch-up time for any incomplete skills work or missed interviews.
